## ðŸš€ Getting started

install all required dependencies with `bun install`

Create a new day with `bun run create <year>/<day>`

Get a solution with `bun run solve <year>/<day>`, or in watchmode: `bun run solve:watch <year>/<day>`.

Run tests `bun test`
